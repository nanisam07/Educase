# Educase React Assignment

A pixel-perfect React implementation of the PopX mobile application UI based on the provided Adobe XD design.

## Features

* Welcome Screen
* Login Screen
* Create Account Screen
* Account Settings Screen
* React Router Navigation
* Responsive Mobile-Centered Layout
* Reusable Components
* Pixel-Perfect UI Design

## Tech Stack

* React.js
* Vite
* React Router DOM
* CSS3

## Installation

Clone the repository:

git clone <repository-url>

Navigate to the project:

cd educase

Install dependencies:

npm install

Run the development server:

npm run dev

## Build for Production

npm run build
